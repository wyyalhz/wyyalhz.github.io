添加live2d看板娘

参考项目

如果你是小白，或者只需要最基础的功能，那么只用将这一行代码加入 html 页面的 head 或 body 中，即可加载看板娘

<script src="https://fastly.jsdelivr.net/npm/live2d-widgets@0/autoload.js"></script>

我是懒人

并非懒人。昨天从六点回到宿舍就就开始信心满满地搞起live2d，参照大佬舍友gyg给的这个项目，因为hexo是静态网站，一开始就直接选用cdn引入，但报错版本不匹配；索性改用npm安装后import，但是hexo不会主动去找node_modules里的包，所以还是报错;于是把.js拷贝到source/js里，想着这样就可以被hexo识别了，但是依然报错找不到……

最后还是用了懒人代码一键导入，真香。

以后找时间再研究一下最开头那个项目，毕竟舍友的项目是基于vue的，而hexo是基于ejs的。

但是舍友又给了一个nb的思路。他把他的vue打包成了几个js文件发我，让我在html里引入，然后就可以直接用了，并且还发我了他的御坂美琴模型和vue项目的源码。这样我以后就可以在vue框架下修改模型，然后再打包成js文件，再引入到hexo里。虽然看起来麻烦了一些，但以后就有vue大佬帮我维护（还可以白嫖御坂美琴模型）了😋

具体做法是：

·注：vue项目存储在D:/live2d/

1.将app.eab99ad8.js和chunk-vendors.45dbdb2.js拷贝到source/js里，并在layout/_partial/head.ejs里引入

    <script defer="defer" src="/js/chunk-vendors.405dbdb2.js"></script>
    <script defer="defer" src="/js/app.eab99ad8.js"></script>

2.在source里新建L2D和Resources文件夹，分别存放live2d模型和御坂美琴的模型(mikoto文件夹)，并在layout/_partial/head.ejs里引入

     <script src="/L2D/live2d.min.js"></script>
     <script src="/L2D/live2dcubismcore.js"></script>
     <script src="/L2D/live2dcubismcore.min.js"></script>

这样就在页面底部成功加载出一个大大的御坂美琴了😀

2025.3.28

给文章添加目录

起因是c++八股内容繁多，文章冗长，为了方便查找和阅读打算添加目录。但是，markdown中目录语法是

- [目录标题](#id)

而后面这个id，如果没有特别声明，vsc和hexo和github都会自动生成一个id，但是自动生成的不一样。例如11章我的标题是

11.References & the Copy-Constructor
vsc生成的id是

- [11.References & the Copy-Constructor](#11-references--the-copy-constructor)

而本地localhost生成的id是

- [11.References & the Copy-Constructor](#11-References-the-Copy-Constructor)

而github生成的id是

- [11.References & the Copy-Constructor](#11-References-the-Copy-Constructor-11)

而而的而且，即使我写对了id，在github page上也无法跳转到对应的标题，暂时没找到原因，故最后手动添加id，这样还能保证本地调试和github的一致性。

添加音乐播放器

参考项目：

BV1VY411o7sW
https://github.com/QingXia-Ela/vue-mini-player
进入github后克隆left-side分支内容到particlex的source下，将release中的css和js分别放入source/css和source/js文件夹下，然后在layout.ejs中引入

    <!-- 音乐播放器 -->
    <!-- .css file path -->
    <link rel="stylesheet" href="/css/index.bundle.css">

    <!-- petite-vue CDN -->
    <script src="https://unpkg.com/petite-vue"></script>
    <!-- .js file path -->
    <script src="/js/index.bundle.js"></script>
    <!-- add songs -->
     <script src="/js/index.music.js"></script>

然后创建index.music.js文件并添加歌曲信息
